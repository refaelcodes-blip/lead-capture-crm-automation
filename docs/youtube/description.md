# YouTube Description

Email / Lead Intake Automation in n8n: Gmail Trigger, LLM Classification, JSON Extraction, Google Sheets, and Telegram

This video shows the full workflow of the ?lead-intake-automation? automation in n8n: from an incoming email or manual webhook request to automatic lead classification, structured data extraction, and sending the result to Google Sheets and Telegram.

## What is demonstrated

- Gmail Trigger for automatically receiving new emails
- Manual Webhook for testing lead submissions through a REST API
- LLM-based classification of incoming messages
- Structured JSON field extraction
- ?Is Lead?? condition routing
- Telegram notification for qualified leads
- Google Sheets logging through a Google Apps Script Web App
- Fallback logic for Gemini / LLM API quota limits
- Workflow testing with a Python SMTP script, webhook requests, and n8n Executions

## Technologies

n8n, Gmail Trigger, Google Sheets, Telegram Bot, Gemini API, REST API, Webhook, Apps Script, Python SMTP testing, Structured JSON Output.
